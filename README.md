# Maaz Momin Angular Showcase

This project has been converted into an Angular-style workspace so the codebase clearly presents itself as an Angular app.

## Stack

- Angular 19 standalone components
- Angular Router
- Reactive Forms
- Tailwind CSS

## Project Shape

```text
src/
  app/
    components/     shared Angular shell pieces
    data/           site content and route data
    pages/          routed Angular page components
    app.routes.ts   Angular route map
    app.config.ts   Angular app providers
  main.ts           Angular bootstrap entry
  styles.css        global Tailwind and site styles
```

## Commands

```bash
npm install
npm start
```

`npm start` runs the Angular dev server.
